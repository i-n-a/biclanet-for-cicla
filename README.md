# Cicla EcoPod – Smart Urban Bike Parking System

## About the Project
EcoPod is a smart, connected bicycle parking system for urban environments, created in collaboration with Cicla. This project combines physical computing, AI/ML prediction models, IoT sensors, and real-time interfaces to improve security, access, and data visibility for daily cyclists in Lisbon.

> 📅 Project Duration: Oct 2025 — Jan 2026

>  Technologies: ESP32, p5.js, Unity, Arduino, Machine Learning (Python), etc.
