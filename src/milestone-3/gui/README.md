# Unity GUI Prototype – Milestone 3

This folder documents the Unity-based GUI and interaction prototype
developed for Milestone 3 of the Cicla EcoPod project.

Due to GitHub file size limitations, the full Unity project (including
Assets, Packages, and ProjectSettings) is provided as a downloadable ZIP.

## Full Unity Project
🔗 Download: [(https://drive.google.com/drive/folders/1_zIl7q734xmbDI-KOI4og99U9i9lVdTN?usp=drive_link)]

## Included in this repository
- Key Unity C# scripts
- Scene files demonstrating:
  - Pod access flow
  - User interaction logic
  - Visual simulation of the EcoPod system

The Unity prototype demonstrates the GUI layer that integrates with
IoT data and ML logic developed in this milestone.
